## Supported

| Item | Docs | Status | Notes |
| --- | --- | --- | --- |
| Shape formatting | `shape-formatting/` | Supported | Solid/gradient/background fills, line color/width/dash, arrow heads, shadow, glow, reflection, blur, soft-edge, line joins, and rotation are marked `[x]`. |
| Shape manipulation | `shape-manipulations/` | Supported | Find, remove, hide, reorder, align, SVG render, and basic shape traversal are marked `[x]`. |
| Primitive shapes | `shape-types/`, `rectangle/`, `ellipse/`, `line/`, `custom-shape/`, `group/` | Supported | Rectangles, ellipses, lines, callouts, connectors, groups, freeform shapes, and shape builder/custom geometry flows are marked `[x]`. |
| Backgrounds | `presentation-background/` | Supported | Solid color, gradient, image backgrounds, and background value reads are marked `[x]`. |

## Not Supported

| Item | Docs | Status | Notes |
| --- | --- | --- | --- |
| Pattern / texture / picture fills | `shape-formatting/` | Not supported | Pattern fill and picture/texture fill remain `[ ]`. |
| 3D shape format | `shape-formatting/` | Not supported | 3D shape format remains `[ ]`. |
| Cross-slide copy / duplicate | `shape-manipulations/` | Not supported | Copy or duplicate a shape to another slide remains `[ ]`. |
| Aspect ratio locking | `shape-manipulations/` | Not supported | Lock/unlock aspect ratio remains `[ ]`. |
| Flip helpers | `shape-manipulations/` | Not supported | Flip shape helpers remain `[ ]`. |
| Preset effects and effective-property reads | `shape-formatting/`, `shape-manipulations/` | Not supported | Preset effects and inherited/effective fill/line/font lookup remain `[ ]`. |
| Callout-specific helpers | `callout/` | Not supported | Chart callout label creation, line styling, and textbox positioning remain `[ ]`. |
| Watermark API | `watermark/` | Not supported | No dedicated watermark API is listed in `TODO.md`. |
| Accessibility features | `presentation-accessibility/` | Not supported | Alt text, reading order, decorative marking, and accessibility reporting remain `[ ]`. |
