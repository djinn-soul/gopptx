# TODO Index

This folder is the working support matrix for the Aspose Slides Python docs dump.

## Structure Summary

- `01` covers core presentation lifecycle and slide-level operations.
- `02` and `03` split text and shape support from the old combined text/shapes index.
- `04`, `05`, and `06` split charts, tables, and SmartArt from the old combined charts/tables/SmartArt file.
- `07` to `11` split media, fonts, notes/security/properties, conversion/import/export, and file-format status from the old media/conversion index.
- `12` tracks advanced limitations and runtime constraints.
- `13` tracks documentation quality and developer-experience gaps.
- `14` tracks missing text features in `gopptx`.
- `15` tracks missing shape features in `gopptx`.
- `16` tracks missing chart features in `gopptx`.
- `17` tracks missing table features in `gopptx`.
- `18` tracks missing image/media features in `gopptx`.
- `19` tracks missing masters/layouts/themes features in `gopptx`.
- `20` tracks missing advanced/unique features in `gopptx`.
- `21` tracks missing core presentation-management features in `gopptx`.
- `22` tracks missing slide-operations features in `gopptx`.
- `23` tracks missing advanced shape-operation features in `gopptx`.
- `24` tracks missing text/formatting features in `gopptx` from screenshot-based audit input.
- `25` tracks missing document-level features in `gopptx`.
- `26` tracks missing export/output features in `gopptx`.
- `27` tracks a consolidated high-priority gap analysis mapped to existing trackers.
