Place optional media fixtures here for `examples/23-media-embed/main.go`.

Expected files:
- `poster.png` (video poster/icon image)
- `sample.mp4` (video media file)
- `sample.wav` (audio media file)
- `sample_ole.bin` (OLE payload bytes)

If these files are missing, the example still runs and generates a valid PPTX,
but it skips media insertion steps.
